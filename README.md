RiftUpPatcher
=============

Patcher for oculus .dll files that adds support of RiftUp! upgrade kit.

HOWTO
=====

1) Start the application
2) Click "Choose File" and find the .dll file of the Oculus library (usually named OculusPlugin.dll)
3) Click "Patch File"
